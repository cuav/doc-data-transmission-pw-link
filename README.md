# 概述

WP\_LINK是一款WIFI转串口模块，只需要通过电脑或者手机的WIFI

即可连接飞控进行调参和设置,通过简单可靠的UDP通讯

使用内置天线距离可达50米\*，外置天线300-500米\*.

通过CUAV地面站Hflight 即可设置WIFI频道、名称及密码

兼容电脑地面站：Mission Planner、QGC

手机地面站：tower、CUAV  Hflight、QGC
